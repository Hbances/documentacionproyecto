Instal·lació de Citadel

Per instal·lar el servidor de correu electrònic Citadel, segueix aquests passos.

Instal·lació del servidor de correu electrònic Citadel. Instal·lem els paquets necessaris per compilar el programari Citadel a la Raspberry Pi.

Compilarem Citadel localment ja que pot haver-hi diversos problemes amb la compilació proporcionada des del repositori Raspbian.

sudo apt install build-essential curl g++ gettext shared-mime-info libssl-dev zlib1g-dev

Iniciar el procés d'instal·lació de Citadel amb l'ordre següent:

curl https://easyinstall.citadel.org/install | sudo bash

Quan se us demani que accepteu els termes i les condicions, premeu Y luego ENTER per continuar.

se us demanarà si voleu que l'script instal·li algun paquet addicional que Citadel pugui necessitar. Novament, premeu I, seguit de ENTER per continuar. El procés d'instal·lació portarà algun temps, per això ara és un bon moment per prendre un cafè mentre el teu Raspberry Pi treballa.

Quan Citadel hagi acabat d'instal·lar-se, haureu de configurar el programari.

us demanarà que establiu un nom d'usuari per al vostre usuari administrador. Per defecte, aquest usuari es dirà admin. Si voleu utilitzar un nom d'usuari diferent per a l'administrador, podeu introduir-lo en aquest pas. En cas contrari, podeu prémer l' ENTER tecla per continuar.

A continuació se us demanarà que introduïu una contrasenya per al vostre usuari administrador. La contrasenya per defecte és citadel.Us recomanem que la canvieu per una altra de més segura.

Necessitarem especificar l'usuari sota el qual s'executarà el programari Citadel.
Per defecte, Citadel intentarà executar-se sota el root usuari si no se n'especifica cap. Per a aquesta opció, assegureu-vos d'escriure “ citadel“. Premeu l' ENTER tecla per confirmar el canvi d'opció.

En aquest pas, podeu introduir una adreça IP perquè el servidor Citadel escolti.

Aquesta opció és útil si només voleu que s'accedeixi a Citadel mitjançant una adreça IP específica. Per defecte, Citadel escoltarà a totes les adreces.


Per a aquesta opció, pressionarem ENTER per mantenir la configuració predeterminada.

En aquest pas és especificar el port on voleu que s'executi Citadel.

Només heu de canviar aquest port per defecte 504 si esteu executant diverses instàncies de Citadel en un servidor.

En aquest pas se us demanarà que definiu el mètode d'autenticació.

A menys que tingueu una necessitat específica per a les altres tres opcions, és més segur deixar aquesta configuració a la 0 opció predeterminada.

Premeu l' ENTER tecla per continuar amb el pas final.

En aquests passos finals, definirem els ports per a la interfície web de Citadel a la nostra Raspberry Pi.

Si no esteu executant un altre servidor web, com Apache, és millor configurar HTTP al port 80 i HTTPS al port 443.
