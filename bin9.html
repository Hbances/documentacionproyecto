Instal·lar i configurar servei dns Bind9

Utilitzarem BIND9 com a servidor DNS per gestionar el domini mailcapa.sytes.net. Això permetrà a la Raspberry resoldre les consultes d'aquest domini i que altres dispositius que estiguin connectats a la mateixa xarxa puguin utilitzar-lo per a la resolució de noms.

Actualitza els repositoris del sistema amb l'ordre següent:

sudo apt update

Instal·la els paquets necessaris per a BIND9 amb l'ordre següent:

sudo apt install bind9 bind9utils bind9-doc dnsutils

Verifica si BIND9 està instal·lat correctament amb aquesta comanda:

sudo systemctl status bind9

Crear la carpeta on estarà la zona:

sudo mkdir /etc/bind/zones

Crear arxiu:

sudo cp db.local /etc/bind/zones/mailcapa.sytes.net.zone

Imatge del procés:



  • NS: Indica que mailcapa.sytes.net serà el servidor DNS del domini.
  • A: Assigna una IP a mailcapa.sytes.net.
  • MX: Defineix el servidor de correu com mail.mailcapa.sytes.net.
