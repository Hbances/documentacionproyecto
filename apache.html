Proyecte amb Raspberry PI 4

Instal·lar i configurar el servei Apache

Utilitzarem Apache com a servidor web per allotjar pàgines estàtiques. Aquestes pàgines mostraran informació sobre l'estat i altres apartats de la Raspberry Pi, a més de permetre l'execució de PHP per crear pàgines dinàmiques.

Instal·la Apache amb la següent comanda:

sudo apt install apache2

Imatge del procés:


Comprova que Apache s'ha instal·lat correctament amb la següent comanda:

sudo systemctl status apache2

Instal·la PHP per permetre l'execució d'arxius PHP a Apache:

sudo apt install php

Instal·la el mòdul necessari perquè Apache pugui executar arxius PHP:

sudo apt install libapache2-mod-php

Totes les pàgines es crearan dins de la ruta següent:

/var/www/html/

Arxiu index.php → Mostrarà informació sobre el sistema operatiu.

Arxiu estat.php → Mostrarà informació sobre l'estat del sistema operatiu.

L'arxiu index.php mostrarà informació sobre el sistema operatiu.


L'arxiu estat.php mostrarà informació sobre l'estat del sistema operatiu, com el temps que porta encès, la càrrega del CPU, etc.
